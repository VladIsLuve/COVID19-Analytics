# COVID19-Analytics
Несколько способов для прогнозирования временных рядов, связанных с темой COVID19

Файлы R являются исходными файлами для запуска интерактивного приложения с аналитикой по заболевшим COVID19. Результат доступен по ссылке: https://vlad-is-love.shinyapps.io/COVID19/ В нём представлена диаграмма последней динамики, а также прогноз, основанный на модели Хольта-Уинтерса.

Файл Jupyter Notebook является проектным заданием с курса от "ВК образование". Он также содержит подобную аналитику, однако для прогноза там использованы модели ARIMA (SARIMAX) и библиотека Prophet, а также идёт прямое сопоставления качества работы разных библиотек.
